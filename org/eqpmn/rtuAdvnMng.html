함체장치관리 IP
PORT
Network Setting
IP Gateway
PORT Subnet Mask
장비의 IP를 설정한다.
Network Setting은 장비 재부팅 후 적용된다.
기본 값
IP 192.168.172.200 Gateway 255.255.255.0
PORT 35344 Subnet Mask 192.168.172.1
Device ID Setting
Device ID
장비의 고유 번호를 설정한다. (범위 0 ~ 65535)
Transmission Method
Transmission Method
신호등 상태정보의 서버 전송 방법을 설정한다.
* 0 : 서버로 전송하지 않음
* 1 : Method1, 신호등 상태 변경 될 경우 자동 전송
* 2 : Method2, 100msec 마다 자동으로 신호등 상태 전송
Server Connection Information Setting
Server IP 1 Server IP 2
Server PORT 1 Server PORT 2
제어기가 Client로 동작 시 서버 IP 주소와 포트 값 설정한다.
Transmission Method가 1 또는 2로 설정된 경우, 설정된 IP 주소로 신호등 상태 값이 전송된다
기본 값
Server 1 192.168.172.200 Port 35000
Server 2 35344 Port 35000
Traffic Signal Request
Traffic Signal Transmission
신호등 상태값의 서버 전송 여부를 설정한다(0 : 전송금지, 1 : 전송)
Device Reset
신호등 상태값의 서버 전송 여부를 설정한다(0 : 전송금지, 1 : 전송)
